Description

A container limits the content width and center it on the screen. It can also contain a Grid.

1180px 910px 720px

Name Default Narrow Very Narrow

Container

A container keeps a distance from the screen edge that can be adjusted. Use smaller padding on mobile devices to maximize space with Responsive.
